# Marv Portfolio

A modern, interactive portfolio website built with Next.js and advanced animations.

## Overview

This project is a personal portfolio showcasing multidisciplinary design skills (Graphic Design, UI/UX, 3D Art, Animation). It features a highly interactive user interface with GSAP-powered animations, including a dynamic intro section with chaotic line movements and a mouse-following glow effect.

## Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [GSAP (GreenSock Animation Platform)](https://gsap.com/)

## Features

- **Interactive Intro Section**:
  - Sequential entrance animations for text and images.
  - "Chaotic" random floating background lines.
  - Smooth mouse-following  shadow/glow effect.
- **Responsive Design**: Optimized for various screen sizes.
- **Custom Navigation**: Side navigation component.

## Getting Started

This project uses **Yarn** as the package manager.


### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2.  Navigate to the project directory:
    ```bash
    cd marv-portfolio
    ```
3.  Install dependencies:
    ```bash
    yarn
    ```

### Running the Development Server

To start the local development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build for Production

To create a production build:

```bash
yarn build
```

To start the production server:

```bash
yarn start
```
